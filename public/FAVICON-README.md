# Favicon Setup Instructions

## Quick Method (2 minutes)

1. **Open the generator:**
   - Double-click: \scripts/generate-simple-favicon.html\
   - Opens in your browser

2. **Download:**
   - Click "Download All Sizes"
   - Saves: studyfrontier-favicons.zip

3. **Install:**
   - Extract the zip file
   - Copy all .png files to \public/\ folder
   - For favicon.ico:
     * Visit https://www.icoconverter.com/
     * Upload favicon-32x32.png
     * Download and save as public/favicon.ico

## Alternative: Online Tool

1. Visit: https://favicon.io/favicon-generator/
2. Settings:
   - Text: "SF"
   - Background: #0ea5e9
   - Font Family: Inter
   - Font Size: 70
   - Shape: Rounded
3. Download
4. Extract to public/ folder

## Files You Need

- favicon.ico (multi-size)
- favicon.svg (? already created)
- apple-touch-icon.png (180x180)
- android-chrome-192x192.png
- android-chrome-512x512.png
- favicon-16x16.png
- favicon-32x32.png

## Verify Installation

After adding files, check:
- Browser tab shows favicon
- Bookmarks show icon
- Mobile "Add to Home Screen" shows icon

Your Next.js app is already configured!
